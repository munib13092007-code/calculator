# Calculator

A basic calculator built with vanilla HTML, CSS, and JavaScript as a front-end development practice project.

## Features

- Addition, subtraction, multiplication, and division
- On-screen number and operator buttons
- Keyboard support (digits, `+`, `-`, `x`, `/`, `=`, and `c` to clear)
- Running display of the current operation and result

## Tech Stack

- HTML5
- CSS3
- JavaScript (DOM manipulation, event listeners)

## Project Structure

```
calculator/
├── index.html      # Markup and button layout
├── calculate.js    # Calculator logic and event handling
└── styles.css      # Styling
```

## Usage

Open `index.html` in any modern browser. Click the on-screen buttons or use your keyboard to enter numbers and operations, then press `=` (or Enter-equivalent button) to see the result. Use **Clear** (or the `c` key) to reset.
